Siemens built Knowledge Fabric, an AI system that automates the software development lifecycle by breaking complex modernization tasks into smaller agent-led steps using a "slicing the elephant" pattern with specialized search, user story, architecture impact, task breakdown, and coding agents. The system was built using Spanner Graph, the Google Agent Development Kit, the Gemini API, Agent Platform, Gemini CLI, and Anthropic Claude Code, and a pilot migrating legacy control panels to web-based interfaces showed significant productivity gains.
